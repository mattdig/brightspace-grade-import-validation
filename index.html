Brightspace Grade Import Verification

Instructions

This tool will verify the contents of a CSV file that you have exported from Brightspace. The CSV file should be follow these rules:

  • It must be a CSV file (not XLSX)
  • The first columns must identify the students with ONLY (one or both):
    • OrgDefinedID
    • Username
  • Do not include any addition indentifying columns for the students (email address, name, etc.)
  • Grade columns headings must end in one of:
    • Points Grade
    • Grade Symbol
    • Text Grade
    • Example: "Midterm Exam Points Grade"
  • Adjusted Final Grade column headings must be:
    • Adjusted Final Grade Numerator
    • Adjusted Final Grade Denominator
  • The last column heading must be: "End-of-Line Indicator"
  • The last column of every row of grade data must be: "#" as this is the end of line indicator
